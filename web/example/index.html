🌐 Weblet

Serveur multi-sites statiques avec CLI – Simple, léger, efficace.

Bienvenue !

Ce site est hébergé grâce à Weblet – un outil open-source pour servir vos sites React, Vue, Vite, Nuxt, etc., facilement depuis un seul serveur.

Fonctionnalités clés

  • ✨ Multi-sites sur ports dédiés
  • 🧩 Gestion via CLI interactive
  • ⚡ Ultra léger (Polka-based)
  • 📦 Zéro dépendance front
